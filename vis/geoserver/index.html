# 开始

下载：http://geoserver.org/release/stable/ (opens new window)

安装解压后运行 bin 文件夹下的startup.bat （linux 为 startup.sh）

打开http://localhost:8080/geoserver/web/ (opens new window)，用户名：admin 密码：geoserver
