# 发布-订阅模式

在发布-订阅模式中，消息的发送方，叫做发布者（publishers），消息不会直接发送给特定的接收者，即间接接收消息，叫做订阅者。

发布者和订阅者不知道对方的存在。需要一个第三方组件，叫做信息中介，它将订阅者和发布者串联起来，它过滤和分配所有输入的消息。

类似报社、邮局和个人的关系，报纸的订阅和分发是由邮局来完成的，报社只负责将报纸发送给邮局。报社就是发布者，个人是订阅者， 邮局就是信息中介
